This function inits, calibrates and changes the clock. 

It enables the 32k\+Hz ref Clock for the Autocalibration and also executes the calibration. Afterwards it enables and changes to the 32\+M\+Hz internal Clock and also disables the 2\+M\+Hz clock. Eventually it Locks the current Clock config.

\begin{DoxyNote}{Note}
The 32k\+Hz Clock remains active.
