It Uses the USARTC1 and:

  • 8 bit character size
  • No parity

§ usart_init_tpuart()

void usart_init_tpuart ( void  )

This Method enables and initializes the USART for the specific purpose to communicate with the TPUART1 Chip.

It Uses the USARTC0 and:

  • 8 bit character size
  • Even parity
  • 1 stop bit
  • 19200 Baud
