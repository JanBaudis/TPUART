§ set_clock_32MHz()

void set_clock_32MHz ( void  )

This function inits, calibrates and changes the clock.

It enables the 32kHz ref Clock for the Autocalibration and also executes the calibration. Afterwards it enables and changes to the 32MHz internal Clock and also disables the 2MHz & 32kHz clocks. Eventually it Locks the current Clock config.
